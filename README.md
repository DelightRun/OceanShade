##安装说明

在Ubuntu下键入以下命令（假设你以搭建好自己的Octopress）

    $ cd /path/to/your/blog/
    $ git clone git@github.com:DelightRun/OceanShade.git .themes/OceanShade
    $ rake "install[greyshade]"
    $ rake generate

其他设置可自行修改`_config.yml`文件
